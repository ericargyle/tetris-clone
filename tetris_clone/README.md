# Tetris Clone (pygame)

A small keyboard-playable Tetris clone for Ubuntu.

## Run

```bash
cd /home/moltbot/clawd/tetris_clone
python3 tetris.py
```

## Controls

- **Left/Right**: move
- **Down**: soft drop
- **Space**: hard drop
- **Up** or **X**: rotate clockwise
- **Z**: rotate counter-clockwise
- **C**: hold piece
- **P**: pause
- **R**: restart
- **Esc**: quit

## Notes

- Uses the system package **python3-pygame**.
- No external assets.
